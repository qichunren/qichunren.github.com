Windows平台应用程序开发

采用 msys2 工具将 Linux 平台上开发的软件移植到 Windows 平台上

pacman -Sy mingw32/mingw-w64-i686-make
pacman -Ss make i686
pacman -Sy git gdb g++

pacman -Sy autoconf automake libtool

Synchronizing package databases

pacman -Syu
